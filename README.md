# Росреестр Парсер

Парсер данных с API Федеральной службы государственной регистрации, кадастра и картографии.

## Требования

- Python 3.12+
- [uv](https://github.com/astral-sh/uv) - пакетный менеджер

## Установка

```bash
uv sync
```

## Запуск

```bash
uv run main.py
```

## Структура проекта

```
fsagov/
├── main.py              # Точка входа
├── parser/
│   ├── __init__.py
│   ├── api_client.py    # Работа с API
│   ├── storage.py      # SQLite хранилище
│   └── models.py       # Модели данных
├── config.py           # Конфигурация
├── pyproject.toml      # Зависимости
├── data/               # База данных
└── parser.log          # Логи
```

## Конфигурация

Параметры находятся в `config.py`:
- `PAGE_SIZE` - размер страницы (по умолчанию 1000)
- `DELAY_BETWEEN_REQUESTS` - задержка между запросами (1.5 сек)
- `TIMEOUT` - таймаут запроса (30 сек)
- `RETRY_MAX` - максимум попыток (3)
